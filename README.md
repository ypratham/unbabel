# Unbabel — Speech-to-Speech Translation Mobile App

Unbabel is a mobile app that translates spoken language in real time, returning natural speech in the target language. It’s designed for quick, seamless conversations across languages.

## Features
- Real-time speech recognition and translation
- Speech output in the target language
- Simple, clean UI optimized for one-handed use
- Multi-language support (list yours here)

## Usage
- Tap the microphone to start recording.
- Select source and target languages.
- The app speaks back the translation.
- Review history or resend as needed.
